# Hospital Management System

A Hospital Management System backend built with Django.

## Features

- User Authentication
- Doctor Management
- Patient Management
- Appointment Booking
- Prescriptions
- Billing
- Dashboard

## Tech Stack

- Python
- Django
- PostgreSQL
- HTML
- Bootstrap

## Installation

1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Run migrations
5. Start the development server
